# Password Field Has No Reveal/Show Option

## Mentions: 1 commenter
## Severity: LOW

## Affected Products
- macOS

## Duration
- Ongoing

## Description
Many macOS password fields lack a toggle to reveal the password:
- Can't verify what you typed
- No "show password" checkbox
- Have to type blind and hope
- Leads to failed login attempts

## User Reports
- "I can't see what I'm typing and keep getting my password wrong"

## Workarounds
- Type password in a text field first, then copy
- Use password manager autofill
- Type very carefully

## Note
This may be a deliberate security design choice, but many users consider it a usability bug.
